# Spawn 3 Friendly Wolves Around The Player

Spawns three tamed (friendly) wolves around a player while they are crouching and holding a bone, with a short per-player cooldown to prevent spam.

Generated by **ModForge AI**.

- Minecraft: `1.21.11`
- Loader: Fabric `0.19.2`
- Fabric API: `0.141.3+1.21.11`

## Build

```bash
./gradlew build
```

The compiled `.jar` will be placed in `build/libs/`. GitHub Actions builds it automatically on push and publishes it to Releases.
